Warbot 3

Logiciel de programmation agent

Warbot 3 est la troisème version de Warbot, une platforme de programmation agent. Warbot est un logiciel à but éducatif pour apprendre la programmation agent. Il se présent de la manière suivante : sur un carte deux équipes ou plus s'affronte grâce à des robots et un systemes de bases et de ressources. Le coté éducatif intervient au moment où l'étudiant doit programmer un comportement agent le plus évolué possible afin de remporter la partie.

Ce projet fut réalisé durant un semestre dans une équipe de quatres personnes avec l'aide de M. Ferber enseignant en programmation agent à l'université de Montpellier.

  • Date : Premier semestre 2014
  • Destinataire : Etudiant en programmation agent Master 1
  • Categorie : Projet Java
